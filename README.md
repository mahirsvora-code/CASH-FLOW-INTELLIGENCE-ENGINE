# Cash Flow Intelligence Engine

Standalone Vorco product for telling you if a real estate project will run out of money and what to do about it.

## What this product does
- Answers go / no-go in the first screen
- Shows the first month cash turns negative
- Shows the funding gap needed to survive
- Explains the root cause of the deficit
- Tests fix options through simulation and compares the result

## Folder map
- `spec.md` for the product definition
- `inputs.md` for the data fields the tool needs
- `outputs.md` for the screens and results it should show
- `logic/` for formulas, assumptions, and scenario rules
- `design/` for layout ideas and screen structure
- `research/` for market notes, user pain points, and examples
- `assets/` for reference files and exports
- `notes/` for working decisions and discussion notes

## Build goal
Create a tool that a developer can use before and during construction to understand whether the project will run out of cash and what to change if it does.
