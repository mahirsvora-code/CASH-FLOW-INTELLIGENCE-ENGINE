# Inputs

## Project setup
- Project name
- Location
- Project type
- Start month
- Total project duration

## Inflow inputs
- Sales collections by month
- Loan drawdowns by month
- Booking advances
- Milestone receipts

## Outflow inputs
- Land payment schedule
- Construction cost by month
- Contractor payment schedule
- Material purchase schedule
- Interest outflow by month
- Approval and statutory costs

## Scenario inputs
- Delay in months
- Construction cost escalation percentage
- Sales slowdown percentage
- Loan delay in months
- Collection delay in months

## Optional inputs
- Opening cash balance
- Minimum cash reserve
- Target profit margin
- Expected absorption rate
